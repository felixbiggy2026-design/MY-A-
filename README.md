# 🤖 Felix AI Chat App

A Flask-based AI chatbot powered by Google Gemini API.

## 🚀 Features

- User registration & login
- Secure authentication (Flask-Login)
- AI chat using Gemini 1.5 Flash
- Chat history stored in SQLite
- Docker support
- Production-ready (Gunicorn)
